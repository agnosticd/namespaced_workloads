= ocp4_workload_tenant_quay - Deploy a Quay Tenant to a shared cluster

== Prerequisites

Quay must have been deployed to the shared cluster with the following variables set:

[source]
----
ocp4_workload_quay_operator_registry_save_admin_credentials: true
----

Example deployment for the Shared Cluster:

[source,yaml]
----
ocp4_workload_quay_operator_channel: stable-3.17
ocp4_workload_quay_operator_registry_namespace: quay
ocp4_workload_quay_operator_registry_hostname: quay
ocp4_workload_quay_operator_registry_admin_password: "{{ common_admin_password }}"
ocp4_workload_quay_operator_registry_save_admin_credentials: true

ocp4_workload_quay_operator_registry_quay_overrides: false
ocp4_workload_quay_operator_registry_quay_overrides_env: true
ocp4_workload_quay_operator_registry_quay_overrides_resources: true
----

== The defaults variable file

* This file ./defaults/main.yml contains all the variables you need to define to control the deployment of your workload.
